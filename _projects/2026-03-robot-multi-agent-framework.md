## Overview

This project presents the development of a configuration-driven multi-agent framework designed to enable robotic task planning starting from natural language instructions.

The objective is to bridge the gap between high-level user commands and deterministic robotic execution by structuring the planning pipeline into modular agents responsible for interpretation, parameter extraction, validation, and execution supervision.

## Architecture

The proposed framework follows a modular multi-agent design where each component performs a specific role within the execution pipeline. A key design principle is the explicit separation between probabilistic reasoning modules and deterministic execution components.

## Main Contributions

- Design of a modular multi-agent architecture
- Planning pipeline for instruction interpretation
- Structured parameter extraction
- Full-plan simulation before execution

## Tools

Python, robotics simulation environments, configuration-driven planning, LLM-based modules.
